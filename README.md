# Gaussian Process Regression for Phonetic Data Analysis using _brms_
This repository includes the material for Gaussian Process Regression for Phonetic Data Analysis using _brms_ (Bürkner, 2017).

The datasets are taken from Chuang, Fon, Papakyritsis and Baayen (2021).

**References:**

Bürkner, P. C. (2017). brms: An R package for Bayesian multilevel models using Stan. _Journal of statistical software, 80,_ 1-28.

Chuang, Y. Y., Fon, J., Papakyritsis, I., & Baayen, H. (2021). Analyzing phonetic data with generalized additive mixed models. In _Manual of clinical phonetics_ (pp. 108-138). Routledge.
